\subsubsection*{template$<$typename T$>$\newline
class Chunk$<$ T $>$}
A template class for managing and processing data in chunks. 

This class provides various methods for splitting data into chunks, including fixed-\/size chunks, overlapping chunks, and predicate-\/based chunking.
