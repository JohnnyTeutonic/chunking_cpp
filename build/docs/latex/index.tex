 A C++ library for flexible data chunking and processing operations.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Parallel chunk processing
\item \doxylink{classChunk}{Chunk} compression (RLE, Delta)
\item Basic chunking operations
\item Overlapping chunks
\item Predicate-\/based chunking
\item Sum-\/based chunking
\item Equal division chunking
\item Sliding window chunking
\item Statistical threshold-\/based chunking
\item Similarity-\/based chunking
\item Monotonicity-\/based chunking
\item Padded fixed-\/size chunking
\item Circular buffer
\item Sliding window
\item Priority queue
\item \doxylink{classChunk}{Chunk} list
\item Statistical operations
\item Random chunk generation
\item \doxylink{classChunk}{Chunk} manipulation utilities
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Quick Start}{Quick Start}}\label{index_autotoc_md2}
